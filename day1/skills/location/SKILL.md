# Location

## Instructions

When using this skill:

1. Call `get_current_location` to get the approximate network location.
2. Explain that the result comes from Amap IP geolocation.
3. Make clear that IP geolocation is approximate and is not a precise GPS address.

## Tools

- get_current_location
